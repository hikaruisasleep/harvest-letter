# 🌾 Harvest Letter

**Harvest Letter** is a typing-driven farming simulator where your keyboard is the primary tool for interaction. Play as Budi, a farmer who performs tasks—from tilling soil to collecting water—by typing specific words and commands.

## ✨ Features

*   **Type-to-Action Gameplay:** Every farming task is integrated with keyboard input. Type to plant, water, and harvest crops.
*   **Interactive Environment:** Explore a pixel-art world with functional elements like the village well and dynamic crop growth.
*   **Time-Based Events:** Manage a daily cycle featuring Quick-Time Events (QTEs) that reward speed and accuracy.
*   **Guided Tutorial:** An intuitive system introduced through text commands to help players learn the core mechanics.

## 🎓 Project Origin

Harvest Letter is a collaborative project developed by **Anomaly Studio**, a team of students from **Universitas Internasional Batam (UIB)**. It explores the integration of typing mechanics into traditional simulation gameplay.
